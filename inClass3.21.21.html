External and embedded styles are applied to this paragraph. Text color from embedded style sheet overrides text color from external style sheet.

Inline styles cause this paragraph to have red text which overrides colors from both the embedded and the external style sheets.

Heading in Arial font is inherited from the div

Paragraph inside a div in Times New Roman font overrides the Arial font of the div. Inherits the green color from the div.

Strong text inside a strong tag will override the normal font weight in the embedded style sheet.

The order of precedence is: External, embedded, in-line, tags.
